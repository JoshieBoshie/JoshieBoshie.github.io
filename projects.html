At UC Berkeley, getting specialized help with computer science courses, especially with lower division courses has been difficult, and we realized that many students ask on Facebook or Piazza for help for private tutors and vice versa. However, it was very difficult for students and tutors to actually contact with one another and discuss what the student needed help with due to the nature of the apps. Our objective of the application is to give students and tutors to interact with each other and give a better interactive interface to manage and discuss what the student needs help with and communicate.

Beehive is an open-source project, and it's primary objective is to bring open access to research, for everyone at UC Berkeley. The app aims to gather all research opportunities in one place, with easy ways to browse or search for projects. In addition, this app reaches out to students and faculty to encourage higher participation in research.

Bearmaps is web mapping application that allows users to find directions from one place to another. The app allows one to search for places to go to, display zoomed in/out maps, and find the shortest path from user location to user-requested location. The application is written in Java using Apache Maven.

DurantLoft is an application created from the idea of digitally connecting church communities. The application was written with Ruby on the Rails framework. It can be used to manage and help administrate a shared meeting space called Durant Loft, with specific features to manage overseeing, inventory, and to more easily schedule meetups with others.

B.Y.O.G. or BuildYourOwnGame is a game engine that generates dungeon based rooms within a given space. It can be used for developing 2D tile based games and has moving entities and obstacles. The engine generates a movable entity that can move, avoid obstacles, and teleport. Multiplayer capabilities are also included in the engine.

An interpreter for a subset of the Scheme language. The interpreter is developed using the Python language which goes through a REPL(Read,Evaluate,Print,Loop) for all the inputs a user gives to the interpreter.
